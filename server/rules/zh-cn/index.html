普通玩家规则

  1. 禁止制作卡服机. (这包括使用命令方块生成大量实体还有其他红石卡服机).
  2. 禁止网络暴力.
  3. 我们要尊重是 LGBTQ+ 的人.
  4. 可以少量口吐芬芳,但要注意分寸.
  5. 禁止一切虚假举报.
  6. 你可以使用外挂启动器,但是禁止使用外挂破坏他人的建筑.
  7. 尊重别人的作品,禁止将他人的作品发布到网上并声称被盗建筑为你本人所做.

管理员(op)规则

  1. 所有上述的规则.
  2. 禁止滥用职权. (这包括随意改变他人游戏模式, 在没有理由的前提下随意将玩家封禁或踢出服务器, 以及随意破坏如主城大厅等公共建筑.)
  3. 永远不要杀掉腐竹. (@ItsDan1el_CN)
  4. 禁止给未填写申请表以及未通过面试的玩家管理员(op)权限.

如果破坏任意一条规则,您将被直接封禁5天.
