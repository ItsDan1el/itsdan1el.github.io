Update Plan - Minecraft 1.20

What are planned to add next update? | Jun 9, 2023

Minecraft 1.20!

Recently, Mojang Studios released it's full release version of Minecraft 1.20. The server is planning to update to 1.20 when the 1.20 version of PaperMC is out.

Updating to 1.20 means that you will be able to use all the new blocks added to 1.20, it would also mean that the resourcepack now will be outdated. We will ask for volunteers to help make the new pack(wait for the notice).

New Rank Lookings!

The server is updating it's style of prefixes to a more modern and simple design, the new designs are shown below.

New Group System!

Players with ranks ≥ Speacialist can create groups.

Here are the things you can do with groups

  • Create a custom suffix for your group (mustn't contain 18+ content)
  • Get a world for your group
  • Create group events
  • Other cool stuffs

Small changes

  • Perfect the rank perms system
  • New spawn
  • Fix known bugs
  • Get a stable ip

The new minigame system is introduced with the first "Arena" mode. This mode is basicly just a pvp mode, but you can choose on how to play it with your opponents. You can get all the things you want in the creative enventory and start fighting with your enemy.

The game is still in early-development stage, so it will keep updating
